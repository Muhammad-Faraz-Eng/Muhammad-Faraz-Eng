- Improved pipeline efficiency and reusability through metadata-driven execution and runtime configuration.

### 2) Retail Data Warehouse Development (Microsoft Fabric)
- Built a retail warehouse foundation to unify sales, inventory, and customer datasets.
- Performed modeling and optimization in Fabric notebooks to support BI and exploratory analytics.

### 3) E-commerce ELT Pipeline (Airflow, dbt, Docker)
- Built a fully containerized ELT architecture using Docker and PostgreSQL with Airflow DAG orchestration.
- Designed the Gold layer Star Schema (3 dimensions and 1 fact table).
- Implemented SCD Type 1 and Type 2 using dbt snapshots.
- Added dbt incremental loading for fact orders and data quality tests across Silver/Gold models.

### 4) Car Sales Data Warehouse (AWS Native)
- Architected a scalable pipeline using AWS Glue, Lambda, and S3.
- Applied Medallion architecture (Bronze/Silver/Gold) and delivered an analytics-ready Star Schema model.

### 5) Spotify Data Engineering Project (Azure)
- Engineered a multi-stage pipeline with Azure Data Factory and Databricks (PySpark).
- Applied Bronze/Silver architecture and SCD-based quality handling.
- Used Delta Live Tables to build a reliable Gold layer for BI consumption.

### 6) Live Data Capture Proof-of-Concept
- Built a proof-of-concept for real-time data capture via direct live API integration.
- Demonstrated ingestion of dynamic streaming-style sources.
